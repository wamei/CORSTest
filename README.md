# CORSTest
iOS/Android WebViewアプリでローカルのhtmlファイルからAPIリクエストを送るサンプルです。

# 動かし方
- クライアント

iOS  
`iOS/CORSTest.xcworkspace`をXCodeで開いてビルドすればOK  
  
Android  
`Android`をAndroidStudioで開いてビルドすればOK

- サーバー

```
cd Server
node test.js
```

[http://localhost:4000](http://localhost:4000/index.html) でブラウザで確認もできます
